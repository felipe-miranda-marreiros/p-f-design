# DataGrid

Tabela de dados baseada em TanStack Table com suporte a ordenação, paginação, filtro global e linha mobile customizada. Integra com `Paper` e `Pagination` para composição completa.

## Usage

```tsx
import { DataGrid } from ".";
import { useReactTable, getCoreRowModel } from "@tanstack/react-table";

const table = useReactTable({
  data,
  columns,
  getCoreRowModel: getCoreRowModel(),
});

<DataGrid table={table} aria-label="Transactions" />

// Com status de loading
<DataGrid table={table} status="loading" aria-label="Loading" />

// Com linha mobile customizada
<DataGrid table={table} renderMobileRow={(row) => <MyMobileRow row={row.original} />} />
```
