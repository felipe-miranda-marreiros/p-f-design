# ProgressBar

Barra de progresso horizontal com variantes `rounded` e `flat`. Suporta segmentos múltiplos com cores individuais e exibição de label percentual.

## Usage

```tsx
import { ProgressBar } from ".";

<ProgressBar value={75} />
<ProgressBar value={50} variant="flat" showLabel />
<ProgressBar
  variant="flat"
  segments={[
    { value: 30, color: colors.secondaryColors.green },
    { value: 20, color: colors.grey[900] },
  ]}
/>
```
