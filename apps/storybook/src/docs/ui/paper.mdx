# Paper

Container de seção com fundo branco, padding interno e bordas arredondadas. Composto por `Paper` e `PaperTitle`. O `PaperTitle` aceita um `rightAction` para botões ou links de ação.

## Usage

```tsx
import { Paper, PaperTitle } from ".";

<Paper>
  <PaperTitle rightAction={<Button variant="tertiary" size="small">View All →</Button>}>
    Transactions
  </PaperTitle>
  {/* conteúdo */}
</Paper>
```
