# Tab

Componente de navegação por abas composto por `Tabs`, `Tab`, `TabPanel` e o helper `tabA11yProps`. Integrado com MUI e estilizado conforme o design system.

## Usage

```tsx
import { Tab, Tabs, TabPanel, tabA11yProps } from ".";

const [value, setValue] = useState(0);

<Tabs value={value} onChange={(_, v) => setValue(v as number)}>
  <Tab label="Overview" icon={<HomeRounded />} iconPosition="start" {...tabA11yProps(0)} />
  <Tab label="Budgets" icon={<PieChartRounded />} iconPosition="start" {...tabA11yProps(1)} />
</Tabs>
<TabPanel value={value} index={0}>Conteúdo overview</TabPanel>
<TabPanel value={value} index={1}>Conteúdo budgets</TabPanel>
```
