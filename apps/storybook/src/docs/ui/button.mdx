# Button

Botão primário do design system com 4 variantes: `primary`, `secondary`, `tertiary` e `destroy`. Suporta estado de loading e disabled.

## Usage

```tsx
import { Button } from ".";

<Button variant="primary">Confirmar</Button>
<Button variant="secondary" loading>Salvando...</Button>
<Button variant="destroy" disabled>Excluir</Button>
```
