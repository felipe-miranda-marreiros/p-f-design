# Pagination

Componente de paginação controlado externamente. Recebe `currentPage`, `totalPages` e `onPageChange`. Suporta ellipsis automático para muitas páginas e é responsivo para mobile.

## Usage

```tsx
import { Pagination } from ".";

const [page, setPage] = useState(1);

<Pagination
  currentPage={page}
  totalPages={10}
  onPageChange={setPage}
/>
```
