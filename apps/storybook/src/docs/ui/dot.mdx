# Dot

Indicador visual circular usado para representar categorias ou status com cor. Recebe qualquer valor de cor válido.

## Usage

```tsx
import Dot from ".";

<Dot color="#277D78" />
<Dot color="#F5C97A" />
<Dot color="#826CB0" />
```

## Default

Dot com a cor padrão do tema (verde).
