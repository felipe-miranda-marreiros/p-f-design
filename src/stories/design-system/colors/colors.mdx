# Colors

Our color palette is organized into four main categories:

- **Beige** — Primary brand colors with neutral warmth
- **Grey** — Neutral colors for text and backgrounds
- **Secondary Colors** — Accent colors for UI elements
- **Others** — Extended palette for specialized use cases

All colors are integrated with the MUI theme and are accessible via `theme.palette`.

---

## Usage

```tsx
import { colors } from "@/app/styles/colors";

// Direct usage
<Box sx={{ bgcolor: colors.beige[100] }} />

// Theme usage
<Box sx={{ bgcolor: "primary.main" }} />
<Typography sx={{ color: "text.primary" }} />
```
