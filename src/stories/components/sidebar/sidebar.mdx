# Sidebar

Navegação lateral responsiva com suporte a minimização, estados de loading/error, e itens configuráveis. Adapta automaticamente entre desktop e mobile.

## Usage

```tsx
import { Sidebar, type SidebarNavItem } from "@/stories/components/sidebar";

const items: SidebarNavItem[] = [
  { icon: <HomeRounded />, label: "Overview", path: "/dashboard" },
  { icon: <SwapVertRounded />, label: "Transactions", path: "/transactions" },
];

<Sidebar items={items} status="idle" />
```
