# DonutChart

Gráfico de rosca SVG para visualizar distribuição de valores por categoria. Suporta tamanho customizado, texto central e múltiplos segmentos com cores do design system.

## Usage

```tsx
import { DonutChart } from "@/stories/components/donut-chart";

<DonutChart
  segments={[
    { value: 338, color: "#277D78", label: "Entertainment" },
    { value: 150, color: "#82C9D7", label: "Bills" },
    { value: 200, color: "#F2CDAC", label: "Dining" },
  ]}
  centerText={{ main: "$338", sub: "of $975 limit" }}
/>
```
