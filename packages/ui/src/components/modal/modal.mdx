# Modal

Diálogo modal composto por `Modal`, `ModalHeader`, `ModalBody` e `ModalFooter`. O header aceita título, descrição e botão de fechar. Controlado externamente via prop `open`.

## Usage

```tsx
import { Modal, ModalHeader, ModalBody, ModalFooter } from ".";

const [open, setOpen] = useState(false);

<Modal open={open} onClose={() => setOpen(false)}>
  <ModalHeader onClose={() => setOpen(false)}>Título</ModalHeader>
  <ModalBody>Conteúdo do modal</ModalBody>
  <ModalFooter>
    <Button variant="primary" fullWidth>Confirmar</Button>
  </ModalFooter>
</Modal>
```
