# TextField

Campo de entrada de texto com suporte a ícones, prefixo, tag de cor e modo select. Baseado no MUI `TextField` com estilo customizado do design system.

## Usage

```tsx
import { TextField } from ".";

<TextField placeholder="Digite aqui" />
<TextField placeholder="Valor" prefix="$" />
<TextField error helperText="Campo obrigatório" />
```
